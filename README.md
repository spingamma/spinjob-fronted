# 🌟 Spinjob - Frontend

¡Hola! Bienvenido al proyecto **Spinjob**. Este es el código de la parte visual de la aplicación (lo que ves en la pantalla). 

Esta guía está diseñada para que puedas instalar y ejecutar el proyecto en tu computadora de la forma más sencilla posible, ¡incluso si no tienes mucha o ninguna experiencia programando! 😊

---

## 🛠️ ¿Qué necesitas antes de empezar?

Para que el proyecto funcione en tu computadora, solo necesitas instalar un programa gratuito llamado **Node.js**.

1. Ve a la página oficial: [Descargar Node.js](https://nodejs.org/es)
2. Descarga la versión que dice **"LTS"** (Recomendado para la mayoría de usuarios).
3. Instálalo como cualquier otro programa (siguiente, siguiente, finalizar).

---

## 🚀 Pasos para encender el proyecto

Una vez que tengas Node.js instalado, sigue estos breves pasos:

### Paso 1: Abrir la carpeta del proyecto
Abre la carpeta `spinjob-fronted` en la que te encuentras ahora mismo. 
Dentro de ella, necesitas abrir una **"Terminal"** o Línea de Comandos:
- **En Windows:** Abre la carpeta donde están estos archivos. Haz clic en la barra de direcciones que está arriba en esa misma ventana, borra lo que está escrito, escribe `cmd` y presiona la tecla **Enter**.
- **En Mac:** Abre la carpeta, haz clic derecho y selecciona "Abrir nueva terminal en la carpeta".

### Paso 2: Instalar las descargas iniciales
En esa ventana negra que se abrió (la terminal), escribe o copia y pega el siguiente comando y presiona **Enter**:

```bash
npm install
```
*(Pídete un café y espera un par de minutos mientras se descargan las piezas necesarias para que el proyecto funcione).*

### Paso 3: ¡Arrancar la aplicación! 🎉
Una vez que termine de cargar el paso anterior, escribe este último comando y presiona **Enter**:

```bash
npm run dev
```

### Paso 4: Ver en tu navegador
Si todo salió bien, en la terminal verás un enlace que generalmente se ve así: `http://localhost:5173/` 

¡Solo tienes que abrir tu navegador de internet (como Chrome, Edge, o Safari), escribir esa dirección y listo! Estarás viendo la aplicación funcionando en tu propia computadora. 🥳

---

## 🛑 ¿Quieres apagarlo?
Si ya terminaste de ver o usar la aplicación y quieres apagarla, vuelve a la ventana negra (terminal) donde escribiste el último comando y presiona las teclas **`Ctrl`** y **`C`** al mismo tiempo. A veces te preguntará si estás seguro, en ese caso presiona la tecla `S` o `Y` y luego Enter.

¡Disfruta Spinjob! ✨
